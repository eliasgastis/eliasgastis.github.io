# eliasgastis.github.io

My personal website — a space-themed showcase of research, projects, publications, and CV.

Plain HTML/CSS/JS, no build step. Hosted on GitHub Pages.

## Structure
- `index.html` — all content/sections
- `css/styles.css` — theme & layout
- `js/script.js` — starfield + interactions
- `assets/` — put `cv.pdf` and images here

## Edit
Everything is hand-editable. Search `index.html` for the section you want and update the text.
